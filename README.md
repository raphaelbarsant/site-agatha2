# Landing Page - Presets Agatha Pinheiro

Landing page estática para venda de presets de fotografia com slider interativo antes/depois.

## Stack

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript Vanilla
- Google Fonts (Playfair Display + Inter)

## Estrutura

```
site-agatha/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── img/
    ├── hero.jpg
    ├── agatha.jpg
    ├── before-1.jpg
    ├── after-1.jpg
    └── ...
```

## Como rodar localmente

1. Abra `index.html` no navegador
2. Ou use um servidor local:
   ```bash
   npx serve .
   ```

## Deploy

### Vercel

1. Crie conta em vercel.com
2. Importe o repositório
3. Deploy automático

### Netlify

1. Crie conta em netlify.com
2. Arraste a pasta do projeto
3. Deploy automático

## Imagens

Substitua as imagens placeholder em `img/` pelas fotos reais da Agatha.

## Personalização

- **Cores**: Edite as variáveis CSS em `css/style.css`
- **Textos**: Edite diretamente no `index.html`
- **Preço**: Busque por "R$ 79,90" e substitua
